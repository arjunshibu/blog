# Blog

Personal blog built with Hugo using the Terminal theme.

## Setup

1. Install [Hugo](https://gohugo.io/installation/)
2. Clone this repo
3. Run `hugo server` to start the dev server

## Production

```bash
hugo
```

This generates static files in the `public/` directory. Deploy the `public/` folder to your hosting service (GitHub Pages, Netlify, Vercel, etc.).
